\section*{Problem Statement}
\addcontentsline{toc}{section}{Problem Statement}
The accurate prediction of \gls{logp} is crucial in various fields, including drug discovery and environmental chemistry. 
\section*{Objectives}
\addcontentsline{toc}{section}{Objectives}
The primary objectives of this project are:
\begin{itemize}
    \item To find a curated dataset of molecules and find whether a fingerprint-based logp estimation is feasible
    \item To evaluate the performance of the implemented models in terms of accuracy and efficiency.
\end{itemize}

\section*{Methodology}
\addcontentsline{toc}{section}{Methodology}
The project will follow these steps:
\begin{enumerate}
    \item Data Collection: Gather a dataset of chemical compounds with known \gls{logp} values. The \gls{ZINC20} database will be used as a primary source for this data.
    \item Fingerprint Generation: The molecules will be converted to fingerprints using multiple algorithms, like Morgan (ECFP4) and MACCS Fingerprints.
    \item Model Development: Train multiple machine learning models like \gls{rf}, \gls{svm} and \gls{nn}
\end{enumerate}
